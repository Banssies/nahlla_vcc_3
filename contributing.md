# 🌿 Contributing to Aether

Thank you for your interest in contributing to Aether.  
This project is rooted in empathy, softness, and thoughtful design, and any contribution should help preserve that spirit.

---

## 🌸 Before You Contribute

Aether is not just about building features.  
It is about building them with care.

When contributing, please keep the following in mind:

- kindness matters
- tone matters
- user experience matters
- emotional impact matters
- simplicity matters

Aether should always feel supportive, not demanding.

## ✨ Ways to Contribute

There are many ways to support this project, including:

- improving documentation
- suggesting feature ideas
- refining language and tone
- designing mockups or interface concepts
- creating prototypes
- helping shape the project philosophy into real tools

Every thoughtful contribution matters, even small ones.

---

## 🌷 Contribution Guidelines

Please try to follow these principles when submitting work:

### 1. Keep it gentle  
The project should reflect calm, respectful interaction.

### 2. Keep it clear  
Avoid unnecessary complexity where a simpler solution works better.

### 3. Keep it human-centered  
Think about how a person would emotionally receive the feature, not just how it functions technically.

### 4. Keep it aligned  
Suggestions should support the existing philosophy of compassionate technology.

---

## 🌼 Suggested Workflow

1. Fork the repository  
2. Create a new branch for your changes  
3. Make your updates  
4. Test and review your work  
5. Submit a pull request with a clear explanation of your changes

---

## 🌙 Writing and Design Tone

If you contribute text, interface ideas, or prompts, aim for language that feels:

- warm
- calm
- respectful
- encouraging
- emotionally aware

Avoid language that feels:

- harsh
- shame-based
- overly robotic
- coldly demanding

## 💻 Example of the Aether Approach

Instead of:

**“You failed to complete this task.”**

Aether-aligned language would be closer to:

**“It looks like today was heavy. Want to try again with one smaller step?”**

That difference is part of the project’s identity.

---

## 🌸 Final Note

Contributing to Aether means helping shape technology that treats people more gently.

Thank you for bringing your ideas, time, and care to the project.
